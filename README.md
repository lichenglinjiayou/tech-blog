# Tech Blog

Personal blog built with [Hugo](https://gohugo.io/) and [PaperMod](https://github.com/adityatelange/hugo-PaperMod) theme.

## Writing

```bash
# Create new post
hugo new content/posts/my-post.md
```

## Local Preview

```bash
hugo server -D
```

## Deploy

Merge PR to `main` → auto-deploy to GitHub Pages.
